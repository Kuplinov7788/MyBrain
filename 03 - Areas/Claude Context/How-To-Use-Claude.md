# Claude ni qanday ishlatish — Emirhan uchun

## Asosiy qoidalar

### 1. Aniq so'ra
- Yomon: "bug bor" → Yaxshi: "Login ekranida Google auth crash qilyapti"
- Yomon: "ekran qil" → Yaxshi: "Profile ekrani: avatar, ism, email, logout"

### 2. Kontekst ber
- "Obsidian dan o'qi" — men loyiha kontekstini tiklayman
- "WeWatch vault dan buglarni ko'r" — buglar ro'yxatini olaman
- Rave papkasida ishlasang — CLAUDE.md avtomatik o'qiladi

### 3. Rejim tanlash
- Oddiy task (1-2 fayl) → oddiy so'ra
- Katta task (5+ fayl) → "Plan mode da qil"
- Bir nechta mustaqil ish → "Parallel qil"

### 4. Memory
- "Memory ga yozib qo'y: ..." — muhim narsani eslab qolaman
- "Eslab qol: ..." — preference saqlash
- "Unut: ..." — noto'g'ri memory o'chirish

## Kuchli buyruqlar
- `/commit` — git commit yaratish
- "Obsidian ga yoz" — nota yaratish
- "WeWatch taskini ol" — Tasks.md dan task olish
- "Plan mode" — katta task uchun reja
